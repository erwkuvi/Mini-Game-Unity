# Unity 01

## Overview

This project is about creating a minigame using 3 different characters with diverse abilities, focusing in the following properties from Unity :

- 3D physics
- Tags and Layers
- Multi-scene workflow
- Interactivity and stage design

![Stage1](https://github.com/erwkuvi/Unity01/blob/main/src_github/stage1.gif)

### Stage1 - Exit The Way!

- Unique movement traits per character
- Cooperative puzzle design
- No wall/infinite jumps
- Color/shape-coded exits
- Completion message on successful exit

![layer-matrix](https://github.com/erwkuvi/Unity01/blob/main/src_github/layer-matrix.gif)

### Stage2

- Characters only interact with white or same-colored platforms
- Color-coded platform filtering using Physics Layers
- Scene chaining: auto-load next stage or loop to Stage1

![Stage2](https://github.com/erwkuvi/Unity01/blob/main/src_github/stage2.gif)

### Stage3

- Teleporters: instant character relocation with clear indicators
- Moving platforms and retro-style timing challenges
- Fast-paced platforming and technical level design

![Stage3](https://github.com/erwkuvi/Unity01/blob/main/src_github/stage3.gif)

### Stage4

- Color-coded switches open matching color doors
- White switches inherit character color
- Dynamic platform color change affecting character accessibility

![Stage4](https://github.com/erwkuvi/Unity01/blob/main/src_github/stage4.gif)
